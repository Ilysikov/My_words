Персональный словарь
==================
Предназначен для самостоятельного внесения новых слов во время изучения иностранного языка. В словаре предусмотрена
возможность тренировки, которая состоит в переводе слов, которые рандомно выбирает из вашего списка. Слова могут иметь
несколько переводов. Также имеется функция учета вашего прогресса.

Требования к окружению
==================
Python 3.7
PyQt5
Для запуска тестов не забудьте настроить конфигурацию "Unittest"
